Tips

  • Speak slowly and clearly in the mic.
  • Make sure your room is quiet with less background noise.
  • The table on the right provides extra feature to the user to help them Transcribe. For Example: Speak Period to put a full stop(.)

About Speech Recognition


Speech Recognition is a website with online Notepad that allows it users to convert any audio to text.

It uses webkitSpeechRecognition from Javascript to transcribe audio to text.

New to Website?

Click on the Mic button to start the Speech Recognition


For the first time only: Your browser will pop up a request for you to allow the site to listen to your mic. Click "Allow".

You can also convert your audio files into text.

There are many tools to ease transcription.
For more information swipe down.

In case if Fiphy is not working make sure you check following things.
The website only works on Google Chrome and Safari Browsers for now. You can use Mobile or PC for that.
